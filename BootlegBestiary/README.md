# Bootleg Bestiary

- An experimental mod that adds some weird new enemies to the game!
- This mod is a BIG WIP, I'm highly inexperienced at making new blender assets, texturing, animating, rigging, etc. So whilst I aim to try and polish these enemies as much as I can - expect lots of imperfections! I'm learning with each new enemy that I make.
- Also bugs. There will be bugs. Please report any you find to me on Discord! (Username skeletogne)

# The Bestiary

## Basic Monsters

### Bathylopod

![alt text](https://i.imgur.com/OT7XlWZ.jpeg)

- A hovering cephalopod that lives inside a salvaged diving helmet. Sprays ink, and attacks by swiping it's tentacles!
- Regular Stages: Verdant Falls, Viscous Falls, Wetland Aspect, Sulfur Pools, Gilded Coast, The Simulacrum (Sky Meadow)
- Modded Stages: Wetland Downpour, Fogbound Lagoon, Sunken Tombs, The Simulacrum (Wetland Downpour and Sunken Tombs)

## Minibosses

### Marrowing

![alt text](https://i.imgur.com/zHiiG1A.jpeg)

- A flying, skull-headed serpent that sweeps terrain with its powerful fiery breath. It can dive-bomb, burrow, and erupt from the ground!
- Regular Stages: Distant Roost (post-loop), Abandoned Aqueduct, Iron Alluvium, Abyssal Depths, Helminth Hatchery, The Simulacrum (Abandoned Aqueduct and Abyssal Depths).
- Modded Stages: Ancient Observatory, The Simulacrum (Ancient Observatory)
